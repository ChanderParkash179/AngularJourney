  6. {{ section2 }}
    1. Angular Directive: An Angular directive is special type of technology that can manipulate the DOM Object.
      Directives can do HTML elements removing html elements from DOM dynamically.
    2. Angular - Directive (Components)
      Directives are Components without a View & Angular Components are Directives, with a View
    3. Angular - Directive Types
      • Component Directive: which is an angular directive with a template view
      • Structural Directive: which can change the DOM layout by adding and removing DOM elements
      • Attribute Directive: which can change the appearance or behavior of an element, component, or another directive.
      • Custom Directive: which can create our custom directive from scratch.
    4. Definitions
      1. STRUCTURAL DIRECTIVE
        • Responsible for the HTML Layout
        • Shape or Reshape the DOM Structure by adding and removing HTML Elements
        • You can Identify by leading * symbol
        • Structural Directives are:
          • *ngFor
          • *ngIf
          • *ngSwitchCase
      2. ATTRIBUTE DIRECTIVE
        • Change the Appearances or the Behavior of a DOM Element
        • Attribute Directives are:
          • ngStyle
          • ngClass
    5. Directives
      • NgFor Directive:
        • we use NgFor Directive to render an Array inside the view
        • NgFor Directive is a Structural Directive
        • With NgFor Directive we can Manipulate the DOM.
      • NgIf Directive:
        • we use ngIf for conditional statments
      • Ng-Template Directive (else condition)
        • ng-template tag is used on the replacement of else condition in angular
        • we need to assign a parameter with '#' in the if clause to redirect it
      • Ng-Style Directive
        • ng-style directive is used for styling in the shape of inline-style to the html tags
      • Ng-Class Directive
        • ng-class directive is used for assigning classes for use in CSS stylesheet for design
